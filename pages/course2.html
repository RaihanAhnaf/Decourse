About Course

Here you will learn how to create a User Interface Design for applications on Mobile. This course is designed for those of you who are just starting out in the UI / UX world or graphic designers in general.

The material discussed is not only limited to teaching how to make a good UI design visually, but here you will also learn the basics of creating a User Interface design. Starting from simple research, making UI designs in software, prototyping, to preparing designs for developers.
